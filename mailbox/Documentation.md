**[MAILBOX]** is a mail inbox application built using React.js and Node.js. The app allows users to view their email messages. The app implements an API layer to fetch and update message data stored in a JSON file named messages. The top bar of the app indicates the number of unread messages and the name of the user. The app is styled using CSS.
